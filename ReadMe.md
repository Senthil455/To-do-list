# Task Manager  A clean, responsive to-do list application built with vanilla JavaScript. Features persistent storage, task filtering, and dark mode support.  ## Features  - Add, complete, and delete tasks - Filter by All / Active / Completed - Persistent storage via localStorage - Responsive design (mobile-friendly) - Dark mode support (system preference) - Keyboard shortcuts (Enter to add, Escape to clear) - Duplicate task prevention - Smooth animations and transitions  ## Usage  1. Open `index.html` in a web browser 2. Type a task in the input field and press Enter or click "Add Task" 3. Click the circle icon to toggle task completion 4. Click the X icon to delete a task 5. Use filter buttons to view All / Active / Completed tasks  ## Project Structure  ``` index.html    ΓÇö HTML structure style.css     ΓÇö Styles and responsive layout script.js     ΓÇö Application logic ```  ## Browser Support  Works in all modern browsers (Chrome, Firefox, Safari, Edge).  ## License  MIT
